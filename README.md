Название: Шапка сайта с Swiper слайдером с эффектом Parallax на Gulp сборке

Стек: HTML, CSS (SCSS), JS (Swiper) 

Задачи: 
1. Спроектировал масштабируемую архитектуру стилей по методологии БЭМ, инкапсулировав всю визуальную логику тем (фоны, градиенты, маски заголовков) внутри SCSS-модификаторов. Реализовал декларативный подход: JavaScript выступает исключительно триггером состояний, что исключает inline-стили, упрощает поддержку кода и позволяет добавлять новые слайды максимально быстро и безопасно без изменения логики скриптов.

Фичи:
1. Создал эффект сквозного скольжения линии для пунктов навигации: анимация появления и исчезновения подчеркивания имеет однонаправленный вектор движения (слева направо), что повысило воспринимаемое качество интерфейса до премиум-уровня без ущерба для производительности.
2. Настроил свойство pointer-events для абсолютно позиционированного блока и его целевых компонентов, что позволило пользователю беспрепятственно взаимодействовать со слайдером на заднем плане.
3. Полностью кастомизировал визуальный стиль генерируемой Swiper полосы прокрутки и бегунка, что создало эффект стильной журнальной верстки, которая наглядно показывает прогресс прокрутки полноэкранных слайдов, не перегружая экран лишними деталями.
4. Разработал кастомную фракционную пагинацию (счетчик слайдов) на базе Swiper API и CSS Grid/Flexbox. Кастомизировал генерируемые плагином элементы, внедрив асимметричную иерархию шрифтов для улучшения UX.
